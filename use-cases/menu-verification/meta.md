# Menu Verification at a Food Delivery Company

**Level:** Intermediate  
**Tagline:** Learn to evaluate AI that checks restaurant menu changes at scale  
**Lessons:** 21  
**Best for:** PMs who've shipped AI features and want a systematic, repeatable eval framework

## What you'll work on

A production-grade AI pipeline that ingests menu change requests from restaurants (price updates, new items, allergen edits) and decides whether to approve or flag each change for human review. You'll analyse real trace logs, write quality standards, design metrics, run red-team exercises, and make ship/hold decisions — all grounded in this one concrete use case.

## Why this use case

Menu verification sits at the intersection of trust, safety, and scale — the exact pressures that make AI evaluation non-trivial. Every lesson's PM decision point maps directly to a judgment call a real AI PM would face.
