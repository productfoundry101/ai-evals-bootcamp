# Building a Conversational Language Tutor

**Level:** Beginner  
**Tagline:** Learn AI evals by building and testing a language-learning assistant  
**Lessons:** Coming soon  
**Best for:** PMs new to AI evaluation who want an intuitive, consumer-product entry point

## What you'll work on

An AI-powered language tutor that holds spoken-style conversations with learners, gives feedback on grammar and vocabulary, adapts to proficiency level, and decides when a learner is ready to advance. You'll evaluate response quality, test for edge cases (wrong language, offensive input, learner frustration), and define what "good tutoring" actually means in measurable terms.

## Why this use case

Language tutors produce open-ended, hard-to-grade outputs — which makes them a perfect sandbox for learning eval fundamentals without needing deep domain expertise. If you can define quality for a language tutor, you can define it for almost anything.
